# よろず支援拠点 面談 相談トピック整理

> 目的：Instagram運用代行（B2B集客特化）でゼロから実績を作るため、よろず支援拠点を「最初のオフライン窓口」として活用する。
> 面談時間は通常60〜90分。聞きたいことを先に出し切ってから、会話しながら優先度を調整するのがおすすめ。

---

## 0. 面談前に伝える「自己紹介3点セット」（最初の5分）

長々と語らず、相手に動いてもらいやすい形で。

1. **何屋か**：「店舗・中小企業向けのInstagram運用代行をやっています」
2. **現状**：「既存アカウントはBtoC向けで、運用代行の見込み客が集まらない。新しく専用アカウントを立ち上げ、最初のモニター実績を作りたい段階です」
3. **何を相談したいか（一言で）**：「最初の1〜3社と接点を持つための、地域での動き方をアドバイスしてほしい」

---

## 1. 顧客開拓・マッチング系（最優先）

よろず支援拠点は「地元の事業者ネットワーク」を持っているのが最大の強み。ここを引き出さないと面談の意味が薄い。

- [ ] **SNS集客に困っている地元の事業者を紹介してもらえるか？**
  （モニター価格・無料相談でもOKと伝える）
- [ ] **商工会議所・商工会・地域の異業種交流会のうち、運用代行と相性が良いのはどれか？**
- [ ] **自分側から「専門家登録」「セミナー講師登録」できる制度はあるか？**
  （地域の創業塾・SNS活用セミナーで講師の口があると一気に信頼構築できる）
- [ ] **過去にSNS関連の相談が来た事業者がいれば、再アプローチ可能か？**
- [ ] **支援拠点のメルマガ・ニュースレターに掲載してもらえる枠はあるか？**

## 2. サービス設計・価格戦略

実績ゼロの状態で「いくらで売るか」は壁になりやすい。担当コーディネーターの過去事例から相場感をもらう。

- [ ] **地元の運用代行の相場感（月額／成果報酬／撮影込み の3パターン）**
- [ ] **モニター価格をいくらに設定すれば「安すぎず・取りやすい」か**
- [ ] **契約期間の設定（3ヶ月縛り？月単位？）の地元事業者の反応**
- [ ] **「投稿代行」だけでなく「DM運用」「広告運用」までセットにすべきか**
- [ ] **自分の強み（◯◯）を打ち出すサービス名・キャッチコピーへの第三者意見**

## 3. 信頼構築のための「武器」作り

ポートフォリオ・実績資料がない状態をどう補うかの相談。

- [ ] **提案書・サービス紹介資料のフォーマット添削をしてもらえるか**
- [ ] **「運用代行の効果」を見せる資料の作り方（数字がない状態で）**
- [ ] **創業計画書・事業計画書のレビュー**（補助金申請にも転用できる）
- [ ] **専門家派遣制度を使って、デザイナー・士業との連携が可能か**

## 4. 補助金・助成金（事業者側に提案できる武器になる）

自分が使うだけでなく、**「顧客の補助金申請を支援できます」が営業トークになる**のが大きい。

- [ ] **小規模事業者持続化補助金**：SNS運用代行費用は対象になるか／直近の採択動向
- [ ] **IT導入補助金**：運用代行は対象になるか／自分がIT導入支援事業者になれるか
- [ ] **自治体独自の販路開拓補助金**（市区町村のもの）
- [ ] **創業補助金・スタートアップ系**：自分自身が使えるもの
- [ ] **申請書類の作成サポートは、よろず側でどこまで見てもらえるか**

## 5. オフライン集客チャネルの広げ方

PDFの戦略マップにあった「異業種交流会・商工会議所」への接続。

- [ ] **おすすめの地元の交流会**（高すぎず、経営層が来るもの）
- [ ] **商工会議所への加入メリット／非会員でも参加できるイベント**
- [ ] **地元の金融機関（信金・地銀）のビジネスマッチング枠**
- [ ] **公的機関主催のセミナーで講師・パネリストになる導線**

## 6. 法人化・契約・労務の壁

運用代行を本格化するときに必ず詰まる事務面。早めに聞いておく。

- [ ] **個人事業 vs 法人化のタイミング**（売上◯◯円が目安？）
- [ ] **業務委託契約書のひな型**を共有してもらえるか
- [ ] **インボイス登録の判断**（事業者向けなので登録の影響大）
- [ ] **著作権・肖像権の取り扱い**（撮影素材・他社投稿の引用）

## 7. 次回アクションを必ず決めて帰る

面談1回で終わらせず、継続関係に持ち込む。

- [ ] **次回面談の予約**（具体的な日程をその場で）
- [ ] **次回までに自分が持ち込む宿題**（サービス資料v2、ターゲット選定リスト など）
- [ ] **担当コーディネーターの専門領域**を聞いて、別分野の相談は別の人に当てる
- [ ] **「実績作りに協力してくれそうな事業者をリストアップしておく」と先方に依頼**

---

## ★ 面談中の振る舞いメモ

- 「教えてもらう」より **「一緒に考えてもらう」スタンス**（コーディネーターは助言が役割で、答えを持ってない場合もある）
- ノートPCより**紙のメモ＋名刺**のほうが会話が弾みやすい
- 「**今日の面談後、◯日以内に動きます**」と宣言して帰ると、次回の温度感が違う
- 1回で全部聞こうとしない。**「顧客紹介の話」と「補助金の話」は別回**に分けるくらいの想定で
