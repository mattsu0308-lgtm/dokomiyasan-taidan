# PC初回セットアップガイド（ブラウザだけで完了）

> 所要時間：**10分**
> 必要なもの：**PC（Windows / Mac どちらでもOK）** と **Google アカウント** のみ
> インストール作業ゼロ。すべてブラウザだけで完結します。

---

## 全体の流れ

```
GitHubでコードを表示
   ↓
Googleスプレッドシートを新規作成
   ↓
Apps Scriptエディタにコードを貼り付け
   ↓
権限承認
   ↓
動作確認（テスト求人を貼り付け）
   ↓
完成！
```

---

## ステップ 1：Google スプレッドシートを新規作成（1分）

### 1-1. ブラウザを開く
推奨：**Google Chrome**（Apps Scriptとの相性が一番良い）。
EdgeでもFirefoxでもOK。

### 1-2. スプレッドシートにアクセス
アドレスバーに入力：
```
sheets.google.com
```
→ Google アカウントでログイン。

### 1-3. 新しいスプレッドシートを作成
画面左上の「**空白のスプレッドシート**」（虹色の「＋」アイコン）をクリック。

### 1-4. ファイル名を変更
左上の「無題のスプレッドシート」をクリックして、
```
農家プロスペクト管理
```
などに変更。

---

## ステップ 2：GitHubからコードをコピー（1分）

### 2-1. このリポジトリの Code.gs を開く

ブラウザの別タブで開く：
```
https://github.com/mattsu0308-lgtm/dokomiyasan-taidan/blob/claude/prepare-consultation-topics-lvFlu/automation/Code.gs
```

### 2-2. Raw（生のテキスト）表示にする

ページ右上の「**Raw**」ボタンをクリック。
→ コードだけのプレーンテキストページが開く。

### 2-3. 全選択してコピー

- **Windows**：`Ctrl + A` → `Ctrl + C`
- **Mac**：`Cmd + A` → `Cmd + C`

これで全コードがクリップボードに入った状態。

---

## ステップ 3：Apps Script エディタにコードを貼り付け（2分）

### 3-1. スプレッドシートに戻る

ステップ1で作った「農家プロスペクト管理」のタブに戻る。

### 3-2. Apps Script エディタを開く

メニューバー：
```
拡張機能 → Apps Script
```
をクリック。

→ 新しいタブで Apps Script のエディタが開く。

### 3-3. デフォルトコードを削除

エディタに最初から書かれている：
```javascript
function myFunction() {

}
```
の **全部を削除**（`Ctrl + A` → `Delete` / `Cmd + A` → `Delete`）。

### 3-4. コピーしたコードを貼り付け

エディタにペースト：
- **Windows**：`Ctrl + V`
- **Mac**：`Cmd + V`

→ 200行以上のコードが貼り付けられる。

### 3-5. 保存

ツールバーの **フロッピーディスクのアイコン**（💾）をクリック、
または `Ctrl + S` / `Cmd + S`。

→ プロジェクト名を聞かれたら「**農家プロスペクト管理**」と入力して保存。

---

## ステップ 4：スプレッドシートをリロード＆権限承認（3分）

### 4-1. スプレッドシートのタブに戻ってリロード

ブラウザの **再読み込みボタン**（または `F5` / `Cmd + R`）。

### 4-2. メニューに「農家リスト」が追加されているか確認

メニューバーの一番右に「**農家リスト**」という項目が増えていればOK。

> 💡 1分待ってもメニューが出ない場合：
> もう一度ブラウザを再読み込み。それでも出ない場合は Apps Script のタブで
> 関数 `onOpen` を選んで「**実行**」ボタンを押す（権限承認も同時に走る）。

### 4-3. シート初期化を実行

メニュー：
```
農家リスト → シート初期化（ヘッダー作成）
```

→ **初回のみ権限承認画面** が出る。

### 4-4. 権限承認の手順（少し画面が怖いが安全）

1. ポップアップ「**承認が必要です**」→「**OK**」または「**権限を確認**」
2. Googleアカウント選択画面 → 自分のアカウントを選ぶ
3. ⚠️「**Google hasn't verified this app**」という警告画面が出る
   - これは「自作スクリプトだから」の警告で、悪意あるコードではない
   - 不安な人は Apps Script タブに戻ってコードの中身を確認してから次へ
4. 画面下の「**Advanced**」（または「**詳細**」）をクリック
5. 「**Go to 農家プロスペクト管理 (unsafe)**」（または「**安全ではないページに移動**」）をクリック
6. 権限一覧の画面 →「**Allow**」（または「**許可**」）

→ 承認が終わると、シートの1行目に緑色のヘッダーが自動で追加される：
```
登録日 | 農園名 | 所在地 | 経営者 | 栽培品目 | 時給 | 募集人数 | 規模感 |
Instagram検索 | Googleマップ | IGフォロワー | 口コミ | 優先度 | ステータス | メモ | 元テキスト
```

これでセットアップ完了！

## トラブルシューティング

### Q1. 「拡張機能」メニューに「Apps Script」がない
A. Google スプレッドシート以外を開いている可能性。`docs.google.com/spreadsheets/` で始まるURLか確認。

### Q2. 権限承認画面で「Advanced」が出ない
A. 言語設定が日本語の場合は「**詳細**」「**安全ではないページに移動**」と表示される。Chromeの設定でブラウザ言語を確認。

### Q3. テスト求人を追加しても農園名が空欄
A. テスト用テキストをそのままコピーできているか確認（半角全角・改行の崩れに注意）。手動で農園名列に入力してもOK。

### Q4. 「農家リスト」メニューが消えた
A. スプレッドシートを完全リロード（`Ctrl + Shift + R` / `Cmd + Shift + R`）。
GASスクリプトがエラーで止まっている場合もあるので、Apps Scriptタブで実行ログ（「実行数」）を確認。

### Q5. 別のPCでも同じシートを使いたい
A. Googleアカウントでログインすれば自動同期。共有メニューから他人を招待もできる。

### Q6. スマホでも使える？
A. **テキストの貼り付けと一覧表示はスマホでも可能**。ただし Apps Script ダイアログはPC推奨。
スマホからは「閲覧・優先度確認・フォロワー数入力」までに留め、追加作業はPCで行うのがスムーズ。
