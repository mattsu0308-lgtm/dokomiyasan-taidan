# 農家プロスペクトリスト自動化ツール

タイミーの求人情報から、農家のプロスペクトリストを **半自動で** 作るための Google スプレッドシート + GAS。

## できること

- タイミーの求人テキストを **コピペするだけ** で項目別に分解
- **Instagram検索URL** と **Googleマップ URL** を自動生成
- 規模感・SNS有無・口コミ評価から **優先度を自動採点**

## 抽出される項目

| 自動 | 項目 | 補足 |
|---|---|---|
| ✅ | 農園名 | 「〇〇農園」「〇〇ファーム」「ワイナリー」「牧場」などのパターン認識 |
| ✅ | 所在地 | 北海道の主要市町村を認識 |
| ✅ | 時給 | 「時給 1,200円」のような表記から数値抽出 |
| ✅ | 募集人数 | 求人数から規模感を推定 |
| ✅ | 栽培品目 | 50種類以上のキーワードマッチング |
| ✅ | 規模感 | 募集人数から大／中／小を判定 |
| ✅ | Instagram検索リンク | クリックで即検索 |
| ✅ | Googleマップリンク | 農園名＋所在地で検索 |
| 手動 | 経営者名 | タイミーに出ない場合が多いので手動 |
| 手動 | IGフォロワー | Instagram訪問後に入力 |
| 手動 | 口コミ評価 | Googleマップ訪問後に入力 |

## セットアップ手順（5分）

### 1. Google スプレッドシートを新規作成

[sheets.google.com](https://sheets.google.com) → 空白で新規作成 →
シート名を「**農家プロスペクト**」などに変更。

### 2. GAS エディタを開く

メニュー：**拡張機能 → Apps Script**

### 3. Code.gs の内容を貼り付け

このリポジトリの `automation/Code.gs` の中身を **全コピーして貼り付け**。

プロジェクト名を「**農家プロスペクト管理**」などに変更して保存（Ctrl+S / Cmd+S）。

### 4. スプレッドシートをリロード

ブラウザの再読み込みボタンを押す。
メニュー欄に「**農家リスト**」が追加される（出ない場合は1分待ってからリロード）。

### 5. シート初期化

メニュー：**農家リスト → シート初期化（ヘッダー作成）**

初回は **権限の承認** を求められる：
- 「権限を確認」をクリック
- 自分の Google アカウントを選択
- 「Advanced」→「Go to (プロジェクト名) (unsafe)」をクリック
- 「Allow」で承認

> ⚠ 警告画面が出るのは「自作スクリプトだから」で、悪意あるコードではない。
> 不安ならこの `Code.gs` の中身を見直してから承認。

## 使い方（1件30秒〜）

### Step 1: タイミーで農園求人を開く

ブラウザ or アプリで気になる求人を開く。

### Step 2: 全選択コピー

Web版なら：求人ページ上で **Cmd+A / Ctrl+A** → **Cmd+C / Ctrl+C**

### Step 3: スプレッドシートに追加

メニュー：**農家リスト → タイミー求人を追加**

開いたダイアログにテキストを貼り付け → **追加する** ボタン。

→ 構造化された行が自動で追加される。

### Step 4: Instagram と Googleマップを開いて補完

追加された行の **IG検索／マップ** リンクをクリック。
- Instagramでフォロワー数を確認 → **IGフォロワー列に入力**
- Googleマップで口コミ評価を確認 → **口コミ列に入力**

### Step 5: 優先度を再計算

5〜10件追加したら、メニュー：**農家リスト → 優先度を再計算**

→ ★★★／★★／★ で自動ランク付けされる。

## 優先度の採点ロジック

| 観点 | 点数 |
|---|---|
| 規模感：大 | +3 |
| 規模感：中 | +2 |
| 規模感：小 | +1 |
| IG未着手（フォロワー数欄に「0」と入力） | +3 |
| IGフォロワー 1,000未満 | +3 |
| IGフォロワー 1,000〜5,000 | +2 |
| IGフォロワー 5,000〜10,000 | +1 |
| 口コミ 4.0以上 | +2 |
| 口コミ 3.5〜4.0 | +1 |

| 合計点 | ランク |
|---|---|
| 7点以上 | ★★★（最優先・直接アプローチ） |
| 5〜6点 | ★★（候補・継続調査） |
| 3〜4点 | ★（保留・余裕あれば） |
| 0〜2点 | -（対象外） |

## トラブルシューティング

### Q. メニューに「農家リスト」が出ない
A. ブラウザを完全にリロード（Ctrl+Shift+R）。それでも出ない場合は GAS エディタで `onOpen` 関数を手動実行（権限承認も同時に通る）。

### Q. 農園名が抽出されない
A. タイミーの求人タイトルが「カフェ◯◯」のような農園以外のパターンの場合、抽出できない。手動で農園名列に入力すればOK。

### Q. 北海道以外の地域に対応したい
A. `Code.gs` の `locPatterns` の配列に市町村名を追加する。

### Q. 栽培品目のキーワードを増やしたい
A. `Code.gs` の `cropKeywords` 配列に追記する。

## 次のステップ（やる気があれば）

- **Instagramフォロワー自動取得**：Instagram Graph API（要事業者申請）。当面は手動でOK。
- **Googleマップ口コミ自動取得**：Google Places API（有料、月数百円〜）。
- **重複検出**：農園名が既に登録済みなら警告を出す。
- **CSVエクスポート**：他ツール連携用。

まずは **手動で20件登録して感触をつかむ** のが先。自動化の追加投資は、ボトルネックを実感してから。
