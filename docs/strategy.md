# 戦略設計：ゴール逆算

## 最終ゴール（3ヶ月後の状態）

Instagram運用代行・コンサル事業者から月3〜5件の問い合わせ、月1〜2件受注。

## KGI/KPIツリー

```
【KGI】月1〜2件 受注
  │
  ├ 受注率 30%想定 →【KPI①】月3〜5件 商談
  │
  ├ 商談化率 50% →【KPI②】月6〜10件 問い合わせ/DM
  │
  ├ DM到達率 0.5% →【KPI③】月1,200〜2,000人のターゲット層リーチ
  │
  ├ ターゲット層比率 10% →【KPI④】月12,000〜20,000総リーチ
  │
  └ 投稿あたり平均リーチ 500 →【KPI⑤】月30〜40投稿（≒1日1〜1.5投稿）
     ＋ プロフィール経由のLP/DM動線
```

### 重要な気づき

1日1〜1.5投稿で十分。量より「ターゲットに刺さるか」が支配的。50投稿/日は過剰。

## 3ヶ月ロードマップ

### Month 1：基盤構築＋仮説検証

| 週 | やること | 完了条件 |
|---|---|---|
| W1 | ナレッジを5軸に再整理／プロフィール・LP・DM導線設計 | プロフィールにLP/DM CTA配置 |
| W2 | 自動投稿パイプライン構築／手動投稿で10件テスト | システム稼働、反応傾向把握 |
| W3 | 自動投稿開始（1日1〜2投稿）／手動オペ30分（コメント返信） | 1週間継続 |
| W4 | インサイト分析、伸びた軸/沈んだ軸を判別 | データに基づく方針修正 |

Month 1 ゴール：フォロワー300〜500、平均リーチ300、DM 1〜2件

### Month 2：当たり軸の深掘り＋関係構築

| 週 | やること |
|---|---|

| W7-8 | 反応くれた人にDMで関係構築／無料相談オファー |

Month 2 ゴール：フォロワー1,000、月3件のDM、初商談1件

### Month 3：刈り取り＋仕組み化

| 週 | やること |
|---|---|
| W9-10 | LP/DMフロー最適化、商談導線強化 |
| W11-12 | 受注ケーススタディを投稿に還元、紹介ループ作る |

Month 3 ゴール：月3〜5件問い合わせ、月1〜2件受注

## 運用原則

1. Month 1は「自動投稿だけ」やらない。必ず生成物を目視レビューしてAI臭・ターゲットズレを早期発見。
2. 量は最小、質は最大。1日1投稿でも刺さる軸が見つかれば月1件受注は十分可能。
3. 計測なきものは改善されない。インサイトは毎週Claudeに分析させ、軸の取捨選択を行う。
